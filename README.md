# Grandpy_bot


## Code architecture
```
├── config.py
├── doc              : Documentation
├── flasklocal       : Flask app content
│   ├── __init__.py
│   ├── classes.py   : Classes of the app
│   ├── static
│   │   ├── css
│   │   ├── img
│   │   └── js
│   ├── templates
│   ├── tests
│   │   ├── test_*.py
│   │   └── test_view.py
│   │   
│   └── views.py     : routes definition
├── Procfile         : Heroku's running config
└── run.py           : the app running script
```
